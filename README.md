# Personal-Finance-Analysis
Personal Finance Tracker Dashboard, Power BI: Analyzed 12-month personal spend across Bills, Food, Living &amp; Travel using Power BI. Tracked ₹280K income vs ₹210K expenses, uncovering seasonal cash flow gaps, a 25% savings rate, and ₹33K in annual coffee spend. Includes daily transaction trends, sub-category breakdown &amp; account-level analysis.
